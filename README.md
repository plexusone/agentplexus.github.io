# AgentPlexus.dev

Source code for the [AgentPlexus.dev](https://agentplexus.dev) website.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Deployed via GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output is written to `docs/` and deployed automatically via GitHub Actions on push to `main`.
